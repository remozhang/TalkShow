# 过年
#### 开场

```
大家好, 我是张雷(主席般挥手致意), 站在这台上, 我感觉自己就像个企业家(傻笑, 拉拉衣领, 观察观众反应, 冷淡的话. 摸摸话筒)你说是吧?话筒
```

#### 阐述主题

```
还没几天就过年了, 今天我们来聊聊新年.新的一年呢, 大家会确立很多很多小目标, (手握拳托举到头顶)  今天开始我再熬夜就是王八蛋, 熬夜伤身体, 然后选择了通宵; 今年我要好好工作, 升职加薪(愣一会..) ;今年我要减掉30斤, 然后刚才使劲往嘴里送菜, 这不是还没过年吗.接着, 回到老家, 亲戚之间走走, 完蛋了, 每逢佳节胖三斤啊!
```

#### 故事一 走亲戚话题

```
小的时候嘛, 过年去亲戚家, 都非常热情, 就是把家里所有能吃的东西都拿出来招待客人
[家里人]:哎, 家里不是还有饺子嘛, 拿出来, 快拿出来, 给客人尝一尝!(从后向前挥手)
但是其实大家都已经吃过饭了, 所以最常见的情况, 就是所有人, 围着我, 看着我端着一碗饺子吃.
[亲戚]:嗨, 你看他吃得多香, 你看, 这韭菜鸡蛋的吧? 闻着就香
[我内心]: 香, 你倒是自己吃啊(扔筷子)
后来我明白了, 为啥走亲戚, 都要带着我, 你们听说过酒桌上劝酒挡酒的吗, 我就是去给他们挡饭的, 你知道, 就是他们一端上来.
[爸]: (温柔)我吃过了, 我吃过了, (转身脸朝下, 挥手拍我肩膀) 给我儿子吃, 去, 儿子, 来, 正长身体, 半大小子吃死老子, 特别能吃, 来, 替爸爸走一个!
[我]: (吃饭状, 眼神朝上, 懵逼)
[爸]: 别光顾着吃啊, 敬, 敬你王叔叔一碗
[王叔]: 不用不用, 谁没吃过饭
[爸]: 谁没有儿子, 来, 替爸爸走一个
我从来没有想过我人生的第一次battle, 是因为一碗饺子, 就吃到最后, 我实在不行了, 就看看我爸
[我]: 爸, 嗝,(做打嗝状), 我今天已经到位了嗷(捂肚子), 这饺子, 好饺, 好饺(竖起大拇指)
[我]: (转向王叔) 没事, 没事, 老王, 我没事, 我没饱, 我就是吃得太快了, 饺子上头, 没事, 不是我吃不下了, 就是这个白菜馅儿跟韭菜馅儿, 它就不能混着吃, 深水炸弹, 劲儿太大.
哎呀, 第二天起来我就特别难受, 感觉失忆了一样.
[我]:哎呀, 我咋什么都记不住呢?
我抬头一看, 我爸看着我
[爸]: 儿子, 没事, 你没事, 你就是昨天吃太多了, 吃饺子撑的, 撑断片了.
```

#### 故事二: 催婚话题

```
回家过年还有一个可怕之处就是被催婚, 你想呀, 就是七大姑八大姨, 哇呀呀, 天天都会在你面前唠叨, 但是你在这里不一样, 他们见不到你, 他们只能通过打电话, 发微信, 以及建立微信群, 在里面@你的方式来催婚, 这种方式呢, 叫做云催婚.
我有个朋友, 他天天接到他妈妈的催婚电话, 内种感觉就跟接推销产品的电话差不多, (看看观众,)挥手, 不是我, 我今年18. (观众不信的话), 哎呀, 过完年19了, 朋友姓名不方便透露啊. 反正王超立每天一接电话就是
[王超立]: 对不起, 我真的不需要!
[王妈]: 不是, 我跟你说, 那个姑娘真的很好的, 很多人选择她, 非常抢手的, 我跟你说
[王超立]: 我说了, 我真的不需要好伐?
[王妈]: 不是, 儿子, 妈妈给你买火车票, 你回来免费体验一个月, 好不好?
[王超立]: 不是, 妈, 你这样, 我挂电话了啊? 
[王妈]: 你这个孩子真是, 那祝你生活愉快!
```
